# Python Semana 02 — Listas y Diccionarios

Ejercicios y proyecto de la semana 2 del roadmap de aprendizaje.
Enfocado en estructuras de datos: listas y diccionarios.

## Contenido

- `Proyecto_semana.py` — Procesador de calificaciones con reporte en consola

## Funcionalidades del procesador

- Calcula promedio general del grupo
- Cuenta aprobados y reprobados (nota >= 6)
- Muestra el mejor y peor estudiante
- Genera reporte detallado por estudiante

## Cómo ejecutar

```bash
python Proyecto_semana.py
```

## Tecnologías
- Python 3.12
- Listas de diccionarios
- Funciones lambda
